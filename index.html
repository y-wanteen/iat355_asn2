IAT 355 AS2 Design Brief

Please open console to check results found for Video Game data sheet and Cereal Brands datasheet.
  1. To find the min and max value of a dimension, we used d3.extent() which finds the min and max value of a dimension and stores it as an array where the first slot is the minimum value and the second slot is the maximum. To print this in the console, we referenced the name of the array and the array position.

  2. To find the overall sum of a dimension, we used d3.sum() which totals all the numeric values found in the dimension. This output a numeric float value which is printed in the console.

  3. To find the average value of a dimension, we used d3.mean() to total all the values in the dimension and then divide by the number of values. This outputs the average value. Some values had to be filtered through using a data.filter() first to get the correct average value, while others did not.

  4. To count the total number of occurences in a dimension based on a specified criteria, we used the data.foreach() function to check each value of the dimension, and then .startsWith() to check if the dimension name or value starts with the specified criteria. If the value started with the criteria, a variable used for counting was incremented by one. The counting variable was then printed to the console once all the items for the dimension were iterated through.
